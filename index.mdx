FluentAvalonia is a control library that ports WinUI 3 controls and the Fluent Design System to [Avalonia](https://avaloniaui.net/), enabling you to build beautiful, native-feeling desktop apps that run across Windows, macOS, Linux, Android, iOS, and WebAssembly. It is published as the `FluentAvaloniaUI` NuGet package.

## Getting started

<Steps>
  <Step title="Install the NuGet package">
    Add `FluentAvaloniaUI` to your Avalonia project using the .NET CLI or NuGet Package Manager.
    ```bash
    dotnet add package FluentAvaloniaUI
    ```
  </Step>
  <Step title="Add the theme to App.axaml">
    Include `FluentAvaloniaTheme` in your application's styles to enable the Fluent Design System.
    ```xml App.axaml
    <Application.Styles>
        <StyleInclude Source="avares://Avalonia.Themes.Fluent/FluentTheme.axaml"/>
        <fa:FluentAvaloniaTheme xmlns:fa="using:FluentAvalonia.Styling"/>
    </Application.Styles>
    ```
  </Step>
  <Step title="Use a control">
    Add any FluentAvalonia control to your views using the `ui` namespace.
    ```xml MainWindow.axaml
    <Window xmlns:ui="using:FluentAvalonia.UI.Controls">
        <ui:NavigationView>
            <ui:NavigationView.MenuItems>
                <ui:NavigationViewItem Content="Home" Tag="home"/>
            </ui:NavigationView.MenuItems>
        </ui:NavigationView>
    </Window>
    ```
  </Step>
</Steps>

<Note>
  FluentAvalonia v2.x targets Avalonia 11.x. Make sure you are on **Avalonia 11.2.5** or later for the latest FluentAvalonia release (v2.3.0). The 1.x branch (Avalonia 0.10.x) is deprecated.
</Note>
